🍕 Pizza Sales SQL Analysis
📊 Project Overview
This project involves a comprehensive analysis of over 48,000 rows of pizza sales data using SQL. The goal was to derive actionable business insights regarding revenue trends, customer behavior, and product performance.

🛠️ Tech Stack
Tool: MySQL Workbench

Language: SQL (Joins, CTEs, Window Functions, Aggregations)

Design: Canva (for Data Storytelling)

📂 Repository Structure
/data: Contains the raw CSV datasets (orders, pizzas, etc.)

Pizza_Sales_Analysis_Queries.sql: The full technical script containing 15 business queries.

Pizza_Sales_SQL_Analysis_Lakshmi.pdf: The final presentation with visual insights.

💡 Key Business Questions Answered
What is the total revenue and order volume?

Which pizza categories and sizes drive the most sales?

What are the peak hours and days for orders?

Who are our "Top 3" revenue-generating pizzas?
